## ⚙️ Using the Application

The interface focuses on simplicity. When you open the application, you see your current language course.

*   **Dashboard:** This screen shows your daily progress and the streak counter.
*   **Lessons:** Click on a lesson icon to start a session. Each lesson consists of matching words, listening tasks, and translation drills.
*   **Profile:** You can view your earned points and completed badges here.
*   **Settings:** Use this menu to change your target language or adjust volume levels.

## 📝 Tips for Success

Consistency matters more than length. Set aside five minutes each day to practice. This habit helps move new words into your long-term memory. 

If you get an answer wrong, the application shows the correct version immediately. Read the explanation to understand your mistake. Repeat difficult words often to improve your retention.

## 🛠️ Troubleshooting

If you encounter issues during installation or usage, try these solutions:

*   **Application won't open:** Right-click the icon and choose "Run as administrator."
*   **Interface looks small:** Check your Windows display settings to ensure your scale is set to 100% or 125%.
*   **Missing sounds:** Check the volume mixer in your taskbar to verify the application is not muted.
*   **Slow performance:** Close other programs that use large amounts of memory, such as web browsers with many open tabs.

## ❓ Frequently Asked Questions

**Is this software free?**
Yes. This is an open-source project available for anyone to use.

**Does this work offline?**
The initial setup requires an internet connection. Most lessons cache automatically, but you need a connection to sync your final score and streak.

**Can I switch languages?**
Yes. Visit the settings menu to select a new language course at any time. Your progress saves automatically under your profile.

**How do I update the software?**
When a new version becomes available on the release page, download the latest installer. Run the file, and it will automatically replace the older version with the new one while keeping your progress data intact.

**Where does my progress data go?**
The application stores your progress locally on your computer. You do not need to create an account to start learning.
